# Examples

本目录用于存放对外说明时可直接引用的示例。

## 当前示例

1. `scan-output.example.json`
- workspace 扫描输出示例

2. `scaffold-output.example.json`
- 项目级脚手架生成输出示例

3. `requirement-plan-output.example.json`
- 普通需求生成 tracking 与 case 的输出示例

4. `multi-repo-plan-output.example.json`
- 多 repo / workspace 项目生成 plan 的输出示例

5. `fine-grained-plan-output.example.json`
- 高复杂度需求被细拆成更多 case 的输出示例

6. `executor-runtime-binding.example.json`
- `.awf/workflow.json` 中 executor runtime binding 的最小配置示例

7. `executor-result.example.json`
- 外部 executor 返回的结构化结果示例
